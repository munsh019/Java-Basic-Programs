# 📘 Java Basic Programs

A collection of simple and fundamental **Java programs** designed for beginners.  
This repository helps new learners understand basic Java syntax, logic building, and common programming concepts.

---

## 📂 Contents

This repository includes programs covering:

- Basic input/output  
- Conditional statements  
- Looping structures  
- String operations  
- Number algorithms  
- Simple logical programs  

Example programs included:

- Factorial calculation  
- Fibonacci series  
- Prime number check  
- Minimum value finder  
- Palindrome check  
- Reverse string/number  
- Remove characters  
- And many more basic Java tasks
